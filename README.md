Giao diện người dùng cho hệ thống LunaWash - Ứng dụng đặt lịch rửa xe thông minh và quản lý khách hàng thân thiết. 

## 🌟 Chức năng chính
* **Giao diện Khách hàng:** Xem điểm tích lũy, hạng thành viên, đặt lịch rửa xe ưu tiên, và đổi điểm thưởng (Voucher, Free wash).
* **Giao diện Admin:** Quản lý hàng chờ, thiết lập quy tắc đổi điểm và quản lý chương trình khuyến mãi.

## 🛠 Yêu cầu hệ thống
* [Node.js](https://nodejs.org/) (Phiên bản v18 trở lên)
* Trình quản lý gói: `npm` hoặc `yarn`
